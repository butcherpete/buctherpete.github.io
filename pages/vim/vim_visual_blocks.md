To insert on multiple lines:

1. Move the cursor to the first letter of the first line.
2. Enter visual block mode (CTRL+v).
3. Press `j` to select to end of list.
4. Press `I`.
5. Type the prefix. For example, `*<space>`
6. Press the `ESC` key.
